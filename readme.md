# FPGA SQUID Emulator

> This project implements a real-time emulator for the Restively Shunted Junction (RCSJ) model based DC-SQUID, specifically targeting **non-hysteretic SQUIDs** and optimized for FPGA deployment.

## Quick Start

### Prerequisites
*   Xilinx Vivado

### Installation & Setup

1.  **Clone the Repository**: Download the project files to your local machine.
2.  **Create Project**: Open Vivado and create a new project.
3.  **Import Source**: Add all files from the `source/rtl` directory into your project.
4.  **Integrate Modules**: Instantiate `SQUID_Group` and `Ib_cal` in your design.

## Architecture

*   **`SQUID_Group`**: The top-level wrapper module for the emulator.
*   **`Ib_cal`**: A dedicated calculation module that works alongside the main emulator to process real-time bias current (`Ib`) values, enabling the system to dynamically respond to varying bias conditions.

## Parameter Calculation

The emulator requires specific configuration parameters based on the SQUID model and readout amplifier gain. C# code for generating these parameters is located in the `source/auxiliary` folder.

### Coefficient Configuration Guide

> [!IMPORTANT]
> Proper scaling of coefficients is critical for the correct operation of the fixed-point arithmetic units.

1.  **General Format**: All coefficients generated by `index_gen` typically use the **Q3.19** fixed-point format.
2.  **Direct Assignment**: Most parameters can be directly assigned to the corresponding ports of `SQUID_Group`.
3.  **Bias Current Dependant Parameters (`cx` & `cy`)**:
    *   The ports `cx_i` and `cy_i` on `SQUID_Group` are dynamic values related to the bias current.
    *   **Calculation**: In the FPGA logic, the base coefficients `cx` and `cy` (from `index_gen`) must be multiplied by the real-time bias current value.
    *   **Result Format**: This multiplication should result in **Q12.19** format values for `cx_i` and `cy_i`.
    *   **Bit Width Customization**: The specific bit width of the bias current input depends on your application's dynamic range.
        *   *Example*: The provided reference code uses a **16-bit unsigned (Q3.13)** format for the bias current. Users should adjust this definition to match their specific requirements.

## Simulation

A complete testbench is provided in the `source/testbench` directory.
*   **Usage**: Run the simulation to observe the real-time solving process of the emulator. It demonstrates the **entire system's calculation workflow and operational mechanism**.It also shows the SQUID's voltage response to a **ramp flux ($\Phi_a$) signal** under a **fixed bias current**.
